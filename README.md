## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

```bash
git clone https://github.com/ashutoshh-jhaa/admin-pannel-react-tsx.git
cd admin-react
npm install
```

#### Development
```bash
npm run dev
```

#### Build
```bash
npm run build
```

#### Preview
```bash
npm run preview
```
